# LocalServerSocket
Create local server socket using c++

#Build steps
1. Using cmake<br />
  cd LocalServerSocket<br />
  mkdir build<br />
  cd build<br />
  cmake ..<br />
  make<br />
2. Using conan<br />
  Install conan in your machine<br />
  conan install . --install-folder build<br />
  conan build . --build-folder build<br />
